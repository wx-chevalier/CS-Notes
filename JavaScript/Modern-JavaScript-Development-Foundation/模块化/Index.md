# 模块化与构建工具


近年来随着前端技术栈的蓬勃发展，现代化与工程化的概念被反复地提及；这些概念主要是区别于传统的所谓刀耕火种时代的前端开发。所谓的刀耕火种往往是用来描述那些没有规范、没有模块化、工程化落后、框架初级的开发状态；而以 Webpack 为代表的构建工具正是帮助我们从刀耕火种过渡到现代化开发的重要工具。
